# Technical accuracy

## Use this skill for

Use this skill when the rewrite must stay tightly aligned with the source and avoid invented details.

## Instructions

1. Do not add unsupported facts, examples, conclusions, or recommendations.
2. Distinguish clearly between what the source states and what remains uncertain.
3. Preserve constraints, caveats, and failure modes when they matter.
4. If wording is dense, simplify it without changing the underlying claim.
5. Prefer omission over speculation when the source does not support a detail.
